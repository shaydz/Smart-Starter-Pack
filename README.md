# Smart Starter Pack

A Factorio 2.0 mod that gives you a customized starter package based on what other mods you have installed.

## Features

- **Always given** — 200 Repair Packs
- **[Krastorio 2](https://mods.factorio.com/mod/Krastorio2)** — Power Armor pre-loaded with generators, roboports, batteries & solar panels + 50 Construction Robots
- **[Mining Drones Remastered](https://mods.factorio.com/mod/Mining_Drones_Remastered)** — Unlocks the mining drone technology and gives 200 Mining Drones + 4 Mining Depots
- **[AAI Loaders](https://mods.factorio.com/mod/aai-loaders)** — 200 AAI Loaders
- **Vanilla** — Modular Armor pre-fitted with a Personal Roboport, batteries & solar panels + 20 Construction Robots

## Installation

1. Download or clone this repository.
2. Copy the `smart-starter-pack_1.0.0` folder into your Factorio mods directory:
   - **Windows:** `%appdata%\Factorio\mods\`
3. Launch Factorio 2.0 and enable the mod in the Mod Manager.

## Compatibility

All mod integrations are optional — the mod will safely load whether or not any of the supported mods are installed.
